## 주요 기능
- **메뉴, 재료 검색** : 메뉴 또는 재료를 검색하면 그에 맞는 레시피들을 불러옴
- **메뉴 추천** : 매일 매일 추천하는 메뉴가 바뀜
- **메뉴명 전달** : 레시피를 클릭하면 레시피의 상세보기가 나옴
- **레시피 상세보기** : 메뉴의 종류, 재료, 영양정보, 레시피, 저감조리법을 알려줌
- **배포** : Github.io로 배포
